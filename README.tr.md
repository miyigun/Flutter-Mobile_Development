# Flutter-Mobile_Development

Bu repo, Flutter framework kullanılarak geliştirilen **mobil uygulama örneklerini, öğrenme denemelerini ve mini projeleri** içermektedir.  
Amaç, Flutter ile Android ve iOS platformlarında modern, hızlı ve ölçeklenebilir uygulamalar geliştirmeyi öğrenmek ve öğretmektir.

---

## Kategoriler

### 1. Animations & Effects
- Animasyon teknikleri
- Görsel efektler
- Kullanıcı deneyimini zenginleştiren örnekler

### 2. Applications & Mini Projects
- Küçük çaplı uygulama örnekleri
- Gerçek hayata uyarlanabilir mini projeler

### 3. Basics & Learning Examples
- Flutter temel kavramları
- Widget kullanımı
- Dart diline giriş örnekleri

### 4. Media & File Handling
- Dosya okuma/yazma işlemleri
- Firebase entegrasyonu
- Medya dosyaları ile çalışma

### 5. Navigation & Structure
- Sayfa geçişleri
- Routing yapıları
- Uygulama mimarisi örnekleri

### 6. Platform & Advanced Features
- Platform kanalları
- Native entegrasyonlar
- İleri seviye Flutter özellikleri

### 7. State Management
- State yönetim teknikleri
- Provider, Riverpod, Bloc gibi paket örnekleri

### 8. UI & Widgets
- Kullanıcı arayüzü tasarımları
- Özel widget geliştirme
- Responsive tasarım örnekleri

---

## Başlangıç

Projeyi çalıştırmak için:

1. Repoyu klonlayın:
   ```bash
   git clone https://github.com/miyigun/Flutter-Mobile_Development.git
2. Klasöre girin:
    cd Flutter-Mobile_Development
3. Bağımlılıkları yükleyin:
    flutter pub get
4. Uygulamayı çalıştırın:
    flutter run


🛠️ Kullanılan Teknolojiler
    Flutter SDK

    Dart

    Android Studio / Xcode

📌 Notlar
    google-services.json gibi gizli dosyalar .gitignore ile koruma altına alınmıştır.

    Her alt klasör bağımsız bir öğrenme modülü veya proje örneği olarak düzenlenmiştir.
