# Flutter-Mobile_Development

This repository contains **mobile application examples, learning experiments, and mini projects** developed using the Flutter framework.  
The goal is to learn and teach how to build modern, fast, and scalable applications for Android and iOS platforms with Flutter.

---

## Categories

### 1. Animations & Effects
- Animation techniques  
- Visual effects  
- Examples that enrich user experience  

### 2. Applications & Mini Projects
- Small-scale application examples  
- Mini projects adaptable to real life  

### 3. Basics & Learning Examples
- Flutter fundamentals  
- Widget usage  
- Introduction examples to the Dart language  

### 4. Media & File Handling
- File read/write operations  
- Firebase integration  
- Working with media files  

### 5. Navigation & Structure
- Page transitions  
- Routing structures  
- Application architecture examples  

### 6. Platform & Advanced Features
- Platform channels  
- Native integrations  
- Advanced Flutter features  

### 7. State Management
- State management techniques  
- Examples with Provider, Riverpod, Bloc  

### 8. UI & Widgets
- User interface designs  
- Custom widget development  
- Responsive design examples

## Getting Started

To run the project:

1. Clone the repository:
   ```bash
   git clone https://github.com/miyigun/Flutter-Mobile_Development.git

2. Navigate to the folder:
    cd Flutter-Mobile_Development

3. Install dependencies:
    flutter pub get

4. Run the application:
    flutter run


🛠️ Technologies Used
    Flutter SDK

    Dart

    Android Studio /Xcode

📌 Notes
    Sensitive files such as google-services.json are protected with .gitignore.

    Each subfolder is organized as an independent learning module or project example.
